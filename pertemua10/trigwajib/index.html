Tipe 1a: Pilihan Ganda (Satu Jawaban Benar)

1.
[Nomor Asli di Web: 3]
Jika $\sin(x) + \cos(x) = \frac{1}{2}$ dan $x$ berada di kuadran IV, maka nilai dari $\sin(x) - \cos(x)$ adalah...
  1. $\frac{\sqrt{7}}{2}$
  2. $-\frac{\sqrt{7}}{2}$
  3. $\frac{\sqrt{5}}{2}$
  4. $-\frac{\sqrt{5}}{2}$
  5. $-\frac{1}{2}$
Pembahasan: Kuadratkan kedua ruas: $(\sin x + \cos x)^2 = (1/2)^2 \to 1 + \sin 2x = 1/4 \to \sin 2x = -3/4$.
Misal $M = \sin x - \cos x$. Maka $M^2 = 1 - \sin 2x = 1 - (-3/4) = 7/4$.
$M = \pm \frac{\sqrt{7}}{2}$. Karena kuadran IV ($\sin < 0, \cos > 0$), maka $M$ negatif.
Jawaban: B
2.
[Nomor Asli di Web: 6]
Jika $A$ dan $B$ adalah sudut lancip, $A+B = 90^\circ$ dan $\cos A = \frac{1}{3}$, maka nilai dari $\sin(2A) - \tan A$ adalah...
  1. $\frac{10\sqrt{2}}{9}$
  2. $0$
  3. $\frac{2\sqrt{2}}{9}$
  4. $-\frac{14\sqrt{2}}{9}$
  5. $-2\sqrt{2}$
Pembahasan: $\cos A = 1/3 \to$ sisi depan $\sqrt{8} = 2\sqrt{2}$.
$\sin A = \frac{2\sqrt{2}}{3}, \tan A = 2\sqrt{2}$.
$\sin 2A = 2 \sin A \cos A = 2 (\frac{2\sqrt{2}}{3}) (\frac{1}{3}) = \frac{4\sqrt{2}}{9}$.
Hasil = $\frac{4\sqrt{2}}{9} - 2\sqrt{2} = \frac{4\sqrt{2} - 18\sqrt{2}}{9} = -\frac{14\sqrt{2}}{9}$.
Jawaban: D
3.
[Nomor Asli di Web: 7]
Diketahui vektor $\vec{u}$ dan $\vec{v}$ dengan $|\vec{u}|=|\vec{v}|=k$. Jika sudut antara kedua vektor adalah $120^\circ$, dan $\alpha$ adalah sudut antara vektor resultan $(\vec{u}+\vec{v})$ dengan vektor $\vec{u}$, maka nilai dari $\sec^2 \alpha$ adalah...
  1. $1$
  2. $2$
  3. $3$
  4. $4$
  5. $\frac{4}{3}$
Pembahasan: Karena panjang vektor sama, resultan membagi sudut apit sama besar.
$\alpha = 120^\circ / 2 = 60^\circ$.
$\sec 60^\circ = 2$. Maka $\sec^2 \alpha = 4$.
Jawaban: D
4.
[Nomor Asli di Web: 9]
Dalam sebuah segi-12 beraturan yang tertulis dalam lingkaran berjari-jari $R$, panjang satu sisinya adalah...
  1. $R\sqrt{2-\sqrt{3}}$
  2. $R\sqrt{2+\sqrt{3}}$
  3. $R$
  4. $R\sqrt{2}$
  5. $R\sqrt{3}$
Pembahasan: Sudut pusat = $360/12 = 30^\circ$. Aturan Cosinus:
$s^2 = R^2 + R^2 - 2R^2 \cos 30^\circ = 2R^2 - R^2\sqrt{3}$.
$s = R\sqrt{2-\sqrt{3}}$.
Jawaban: A
5.
[Nomor Asli di Web: 10]
Sebuah satelit mengorbit bumi dalam lintasan melingkar pada ketinggian tertentu sehingga ia menyelesaikan satu orbit penuh ($360^\circ$) dalam 90 menit. Jika seorang pengamat di stasiun bumi melihat satelit tersebut melintasi busur langit sebesar $60^\circ$ dari barat ke timur, berapa lama waktu yang dibutuhkan satelit untuk melintasi busur tersebut?
  1. $10$ menit
  2. $12$ menit
  3. $15$ menit
  4. $20$ menit
  5. $30$ menit
Pembahasan: Kecepatan sudut konstan. Waktu total orbit ($360^\circ$) = 90 menit.
Waktu tempuh busur $60^\circ = \frac{60}{360} \times 90$ menit = $\frac{1}{6} \times 90 = 15$ menit.
Jawaban: C

6.
[Nomor Asli di Web: 13]
Tentukan nilai dari $\sin \theta + \cos \theta$.
(1) $\sin\theta \cos\theta = -\frac{12}{25}$.
(2) $\theta$ berada di kuadran II.
  1. A
  2. B
  3. C
  4. D
  5. E
Pembahasan: $(\sin + \cos)^2 = 1 + 2(-12/25) = 1/25$. Maka $\sin + \cos = \pm 1/5$.
Kuadran II saja tidak cukup menentukan tanda karena tergantung besar mana antara $|\sin|$ dan $|\cos|$.
Jawaban: E
7.
[Nomor Asli di Web: 14]
Berapakah tinggi sebuah gedung?
(1) Dari titik A di tanah, sudut elevasi ke puncak gedung adalah $45^\circ$.
(2) Dari titik B, yang berada 20 meter lebih dekat ke gedung dari titik A, sudut elevasi ke puncak adalah $60^\circ$.
  1. A
  2. B
  3. C
  4. D
  5. E
Pembahasan: (1) Hanya memberi rasio tinggi:jarak = 1:1. (Tidak Cukup)
(2) Hanya memberi rasio tinggi:jarak = $\sqrt{3}:1$. (Tidak Cukup)
Gabungan: Sistem persamaan linear dua variabel ($h = x+20$ dan $h = x\sqrt{3}$) bisa diselesaikan.
Jawaban: C

8.
[Nomor Asli di Web: 16]
Titik $P(-5, 12)$ terletak pada sisi terminal sudut $\theta$ dalam posisi standar. Manakah pernyataan berikut yang benar?
Pembahasan: $x=-5, y=12 \to r = \sqrt{25+144}=13$. (A Benar)
$x$ negatif, $y$ positif = Kuadran II. (E Benar)
$\cos \theta = x/r = -5/13$. (B Benar)
$\tan \theta = y/x = -12/5$. (C Benar)
$\csc \theta = r/y = 13/12$. (D Benar)
Jawaban: A, B, C, D, E
9.
[Nomor Asli di Web: 18]
Manakah nilai $x$ dalam interval $[0, 2\pi]$ yang memenuhi persamaan $2\cos^2(x) + 3\sin(x) - 3 = 0$?
Pembahasan: $2(1-\sin^2 x) + 3\sin x - 3 = 0 \to 2\sin^2 x - 3\sin x + 1 = 0$.
$(2\sin x - 1)(\sin x - 1) = 0$.
$\sin x = 1/2 \to x = \pi/6, 5\pi/6$.
$\sin x = 1 \to x = \pi/2$.
Jawaban: A, B, C
10.
[Nomor Asli di Web: 26-30]
Gunakan informasi berikut untuk menjawab soal tabel.
Untuk sebuah sudut $\theta$ di kuadran IV, diketahui $\cos \theta = \frac{3}{5}$. Tentukan apakah pernyataan-pernyataan berikut "Benar" atau "Salah".
Pernyataan Benar Salah
1. Nilai $\sin \theta = -\frac{4}{5}$. ✔
2. Nilai $\tan(2\theta) = \frac{24}{7}$. ✔
3. Nilai $\sin(\frac{\theta}{2}) = -\frac{1}{\sqrt{5}}$. ✔
Pembahasan: 1. Kuadran IV, sin negatif. $\sin \theta = -4/5$. (Benar)
2. $\tan \theta = -4/3$. $\tan 2\theta = \frac{2(-4/3)}{1-(-4/3)^2} = \frac{-8/3}{-7/9} = 24/7$. (Benar)
3. $\sin(\theta/2) = \pm \sqrt{\frac{1-3/5}{2}} = \pm \frac{1}{\sqrt{5}}$.
Cek kuadran $\theta/2$: Jika $270 < \theta < 360$, maka $135 < \theta/2 < 180$ (Kuadran II).
Di Kuadran II, sinus positif. Harusnya $+1/\sqrt{5}$. (Salah)
